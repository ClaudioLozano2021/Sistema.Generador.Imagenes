# Sistema Generador de Imágenes

Este proyecto es una aplicación web construida con **ASP.NET Core MVC** destinada a servir como base para un sistema generador de imágenes.

## Requisitos

- [SDK de .NET 9](https://dotnet.microsoft.com/download)

## Ejecutar la aplicación

1. Restaurar paquetes y compilar el proyecto:
   ```bash
   dotnet build
   ```
2. Ejecutar la aplicación:
   ```bash
   dotnet run --project Sistema.Generador.Imagenes
   ```

La aplicación se ejecutará por defecto en `https://localhost:5001`.

## Estructura del repositorio

- **Sistema.Generador.Imagenes/**: código fuente principal de la aplicación.
- **docs/**: documentación adicional. Actualmente incluye detalles del `HomeController`.
